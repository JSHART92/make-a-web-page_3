Important Things to Remember

Lesson 1: The Basics of the Web and HTML

The World Wide Web

The web is a very amazing thing. It is made up of billions of pages with so many different types of information. When someone goes on www.facebook.com, their computer browser sends a HTTP Request to a server. The server finds the correct page and then relays it back through the chain onto the browser and in front of the person.

HTML
HTML stands for Hypertext Markup Language. HTML is the basis for what most of the webpages are made of. It is made up of text content markup, references to other documents and links to other pages. The current iteration is HTML 5.

Tags and Elements

The HTML documents are created using elements. We tell the web browsers the type of elements by using HTML tags.
Here are a list of some of the more commonly used tags.

Anchor- You can create an anchor tag to link a webpage

Break- A void tag. You can create an inline page break.

Emphasis-This tag is used to create italics.

Image- This tag is used if you want to use an image.

Paragraph- Use this when you want to create an invisible box around a group of text.

Computers Are Only As Smart As You Make Them

Computers only work given a series of very rigorous instructions. If they aren't followed to the letter then you won't get the results you are looking for.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them. Inline elements have no width or height and are just there.

Lesson 2: Creating a Structured Document with HTML

Using Developer Tools in the browser

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them. Inline elements have no width or height and are just there. You can find the developer tools option located under the hamburger menu in the toolbar. Select more tools and then developer tool. You can see the HTML running in realtime!

HTML's tree like structure

HTML is made up like a family tree. You can have HTML elements inside of other HTML elements. You can have boxes that are within boxes and so forth. The DOM or Document Object Model is the tree like structure that translates the structure.

HTML and Indentation

You can see the indentations on the left side of the code. The further indented the code is the more nested it is within the box.

Text Editors

Text editors are a great way to code for HTML. Several different kinds exist, but for the purposes of this class we use Scratchpad, Codepen and Sublime Text. They range in difficulty of use going from the easiest(Scratchpad) to the hardest(Sublime Text)

Other Notes

HTML is the structure in which a webpage is written. CSS or a Cascading Style Sheet is the style of the webpage. Javascript is also used to create the interactive components of the page.

Lesson 3: Adding CSS Style to HTML Structure

What is CSS?

As mentioned previously, CSS is a Cascading Style sheet. CSS is code written to control the style of the HTML elements.

Three Ways to Include CSS

1.) Write CSS in the head of your HTML 2.) Link your HTML to a separate CSS File 3.) Write your style inline with your HTML

Avoid Repeating Code

It is best not to repeat code. Not only is it harder to read, it can also cause errors in the program which the programmer has to find.
